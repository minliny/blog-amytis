# Binary Search

> 维护左右边界并在有序数组中二分定位目标值。

## 代码区

```ts
function search(nums: number[], target: number): number {
  let left = 0;
  let right = nums.length - 1;

  while (left <= right) {
    const mid = left + Math.floor((right - left) / 2);
    if (nums[mid] === target) return mid;
    if (nums[mid] < target) left = mid + 1;
    else right = mid - 1;
  }

  return -1;
}
```

## 解题思路

维护左右边界并在有序数组中二分定位目标值。

## 复杂度分析

- 时间复杂度：O(log n)
- 空间复杂度：O(1)

## 优化点

- 统一边界收缩规则，避免死循环。
- 用测试覆盖目标不存在与单元素场景。
