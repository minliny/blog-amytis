# Minimum Window Substring

> 先遍历字符串并维护滑动窗口，再用 Map 记录每个字符最近出现的位置；如果当前字符仍在窗口内，就更新左边界收缩窗口，否则继续扩展右边界并更新最长长度。

## 代码区

```ts
function minWindow(s: string, t: string): string {
  const need = new Map<string, number>();
  for (const ch of t) need.set(ch, (need.get(ch) ?? 0) + 1);

  const window = new Map<string, number>();
  let valid = 0;
  let left = 0;
  let start = 0;
  let length = Number.POSITIVE_INFINITY;

  for (let right = 0; right < s.length; right++) {
    const inChar = s[right];
    if (need.has(inChar)) {
      window.set(inChar, (window.get(inChar) ?? 0) + 1);
      if (window.get(inChar) === need.get(inChar)) valid++;
    }

    while (valid === need.size) {
      if (right - left + 1 < length) {
        start = left;
        length = right - left + 1;
      }
      const outChar = s[left++];
      if (need.has(outChar)) {
        if (window.get(outChar) === need.get(outChar)) valid--;
        window.set(outChar, (window.get(outChar) ?? 0) - 1);
      }
    }
  }

  return Number.isFinite(length) ? s.slice(start, start + length) : "";
}
```

## 解题思路

先遍历字符串并维护滑动窗口，再用 Map 记录每个字符最近出现的位置；如果当前字符仍在窗口内，就更新左边界收缩窗口，否则继续扩展右边界并更新最长长度。

## 复杂度分析

- 时间复杂度：O(n)
- 空间复杂度：O(k)

## 易错点

- 重复字符出现在窗口外时，左边界不能回退
- 更新最长长度前要先完成窗口边界调整

## 优化点

- 窗口右移时同步更新字符最近位置，避免左边界回退
- 发现重复字符后用 max(left, lastIndex + 1) 收缩窗口，避免重复计数
