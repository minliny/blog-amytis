# Phase 4 API Delivery Smoke

> This smoke article validates the GitHub API delivery route without relying on local git clone, commit, or push.

## Verification Scope

- Delivery branch is prepared from `main`
- Target content file is written under `content/posts`
- Pull Request is created or reused against `main`
- `main` remains untouched throughout the verification
