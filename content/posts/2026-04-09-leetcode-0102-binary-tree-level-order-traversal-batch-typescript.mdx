# Binary Tree Level Order Traversal

> 先把根节点加入队列并按层遍历二叉树，再在每一层循环中依次取出节点并收集值；如果左子树或右子树存在，就把对应子节点继续入队，否则结束当前层。

## 代码区

```ts
function levelOrder(root: TreeNode | null): number[][] {
  if (!root) return [];
  const result: number[][] = [];
  const queue: TreeNode[] = [root];
  while (queue.length) {
    const size = queue.length;
    const level: number[] = [];
    for (let i = 0; i < size; i++) {
      const node = queue.shift() as TreeNode;
      level.push(node.val);
      if (node.left) queue.push(node.left);
      if (node.right) queue.push(node.right);
    }
    result.push(level);
  }
  return result;
}
```

## 解题思路

先把根节点加入队列并按层遍历二叉树，再在每一层循环中依次取出节点并收集值；如果左子树或右子树存在，就把对应子节点继续入队，否则结束当前层。

## 复杂度分析

- 时间复杂度：O(n)
- 空间复杂度：O(n)

## 易错点

- 空树输入应直接返回空数组
- 按层遍历时不能在循环中动态读取 queue.length 作为当前层大小

## 优化点

- 层序遍历时先固定当前层节点数量，再维护队列顺序，避免下一层节点混入本层结果
- 节点出队后再判断左右孩子是否存在，避免空节点进入队列
