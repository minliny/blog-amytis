# Two Sum

> 先遍历数组并用哈希表记录已访问元素的位置，再判断目标差值是否已经出现；如果 Map 中存在匹配值就直接返回索引，否则插入当前数字和下标继续遍历，整体查找保持 O(n)。

## 代码区

```ts
function twoSum(nums: number[], target: number): number[] {
  const seen = new Map<number, number>();
  for (let i = 0; i < nums.length; i++) {
    const need = target - nums[i];
    if (seen.has(need)) return [seen.get(need) as number, i];
    seen.set(nums[i], i);
  }
  return [];
}
```

## 解题思路

先遍历数组并用哈希表记录已访问元素的位置，再判断目标差值是否已经出现；如果 Map 中存在匹配值就直接返回索引，否则插入当前数字和下标继续遍历，整体查找保持 O(n)。

## 复杂度分析

- 时间复杂度：O(n)
- 空间复杂度：O(n)

## 易错点

- 命中补数后应立即返回，避免后续插入覆盖正确索引
- 存在重复值时要保证返回的是之前保存的下标和当前下标

## 优化点

- 使用 Map 存储数值到索引的映射，避免每次重复查找
